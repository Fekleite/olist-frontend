<h2>🚀 Como utilizar:</h2>

Clonando o repositório:

```
$ git@github.com:Fekleite/olist-frontend.git
```

Instalando as dependências:

```
$ cd olist-frontend
$ npm install
```

Iniciando a aplicação:

```
$ npm start ou yarn start
```
